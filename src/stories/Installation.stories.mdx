## Getting Started

### Install SnappyLib (Cài đặt snappyLib):

<Canvas>
  <>
    <h4>1. Npm install snappy-lib react react-dom</h4>
    <div style={{ color: 'red' }}>
      Note:snappy-lib currently requires React 16.8 or higher.
    </div>
    <h4>2. Include Css (Thêm CSS vào file app.js của bạn)</h4>
    <div>{`import 'snappy-lib/dist/snappy.css';`}</div>
  </>
</Canvas>

### Import components (Thêm thành phần vào dự án của bạn):

<Canvas>
```jsx
  <>
    import React from 'react';
    import { SnyButton } from 'snappy-lib';

    export default (props) => {
      return <SnyButton type="danger">Danger!</SnyButton>;
    };

  </>
```
</Canvas>
