### Snappy Lib is a [React](https://reactjs.org) component library for [Snappy Express](https://snappy.vn)

## Getting Started

### Install SnappyLib:

<Canvas>
  <div>{`npm install snappy-lib react react-dom';`}</div>
  <div style={{ color: 'red' }}>
    Note:snappy-lib currently requires React 16.8 or higher.
  </div>
</Canvas>

### Include Css

Either:

1. Import css in your application code:

<Canvas>
  <div>{`import 'snappy-lib/dist/snappy.css';`}</div>
</Canvas>

2. or include Bootstrap from a CDN URL in your HTML layout:

<Canvas>
  <div>
    {`<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.0/dist/css/bootstrap.min.css"
  />`}
  </div>
</Canvas>

### Import components:

```jsx
<>
  import React from 'react';
  import { SnyButton } from 'snappy-lib';

  export default (props) => {
    return <SnyButton type="danger">Danger!</SnyButton>;
  };
</>
```
